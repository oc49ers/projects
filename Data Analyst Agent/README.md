# Data

This workspace contains a custom VS Code Copilot agent designed to act as Data. The agent focuses on converting data into actionable insights, supporting business decisions, and delivering both Markdown and JSON artifacts.

## Files
- `.copilot-instructions.md` — workspace-level Copilot behavior guidance
- `data-analyst-agent.agent.md` — agent persona, responsibilities, and behavior rules
- `data-analyst-agent.prompt.md` — formal agent prompt, artifact templates, and use instructions

## Capabilities
- Collect and validate data from business and operational sources
- Create data analysis reports, KPI summaries, and forecasts
- Provide clear recommendations backed by metrics
- Generate outputs in both Markdown and JSON
- Ask clarifying questions about data sources, context, and objectives
- Support other agents with evidence-based insights

## How to use
1. Open the workspace in VS Code.
2. Start a Copilot chat and select or invoke Data.
3. Describe your data question, business problem, or analysis need.
4. Ask the agent for a report, KPI summary, forecast, or recommendation.

## Best practice
- Provide clear business context and the decision you want to support.
- Include relevant data sources, metrics, and timeframes.
- Request both summary recommendations and follow-up analysis steps.
