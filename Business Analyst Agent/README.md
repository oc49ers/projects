# Elon - Business Analyst Agent

This workspace contains a custom VS Code Copilot agent designed to act as a Business Analyst, bridging business stakeholders and technology teams.

## Files
- `.copilot-instructions.md` — workspace-level Copilot behavior guidance
- `business-analyst-agent.agent.md` — agent persona, responsibilities, and behavior rules
- `business-analyst-agent.prompt.md` — prompt definitions, artifact templates, and use instructions
- `sample-conversation.md` — example conversation flow for using the agent

## Agent capabilities
- Evaluate operations and identify improvement opportunities
- Gather requirements and analyze business data
- Streamline processes and recommend efficiency solutions
- Create executive memos for leadership communication
- Drive data-driven decisions with supporting analysis
- Generate artifacts in both Markdown and JSON formats

## How to use
1. Open the workspace in VS Code.
2. Start a Copilot chat and select or invoke the Business Analyst Agent.
   - If you have a Copilot Chat pane, choose the agent from the agent selector.
   - If prompted, type a direct request such as: `Use Elon to analyze our operations.`
3. Describe your business problem, operational challenge, or analysis need.
4. Ask the agent for an executive memo, requirements analysis, or process recommendations.

## Best practice
- Provide detailed business context, stakeholder information, and operational data upfront.
- Include current processes, pain points, and success metrics when possible.
- Use the executive memo format for leadership communications.
- Request both qualitative analysis and quantitative data support.
